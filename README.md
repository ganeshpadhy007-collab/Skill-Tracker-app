# 🎯 BCA Skill Tracker

A web app for BCA students to track learning progress, maintain daily streaks, and calculate "Job Ready Score" for placements.

### 🚀 Live Demo
🔗 https://ganeshpadhy007-collab.github.io/Skill-Tracker-app/


### ✨ Features
- **Subject-wise Progress**: Track DSA, Web Dev, Java, DBMS with custom targets
- **Visual Progress Bars**: See % completion for each skill instantly
- **Job Ready Score**: Auto-calculated score based on all skills - know when you're placement ready
- **Add Custom Skills**: BCA syllabus ke alawa bhi skills add kar sakte ho
- **Data Persistence**: LocalStorage se data save hota hai, refresh pe delete nahi hoga
- **Mobile Responsive**: Phone me bhi easily use kar sakte ho

### 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript
- **Styling**: Tailwind CSS CDN
- **Charts**: Chart.js [planned]
- **Deployment**: GitHub Pages / Vercel
